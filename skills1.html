Welcome to Skills Challenges 1.0

This section invites you to complete some challenges. For all challenges, you should use Brackets (or an equivalent text-based editor) to create any HTML and CSS files.

All files should be placed in your Google Folder that starts with "WWW_FY2..."

Skills Challenge 1

Create a simple HTML file that displays your heading, a paragraph and an image.

This file should be saved as sc1.html and saved in your "WWW_FY2..." Google folder.

Sample Page

Specifications

  1. The page must have a header tag (eg. h1 tag).
  2. The page must have a paragraph (eg. p tag)
  3. The page must have an image (img tag)
  4. The page must have HTML tag to start and end the file properly.
  5. The page must be called "sc1.html"
  6. The page might have a body and head tags
  7. The page might have a title (title tag)

Skills Challenge 2

Create a second HTML file that displays a heading, a paragraph and a list.

This file should be saved as sc2.html and saved in your "WWW_FY2..." Google folder.

Sample Page

Specifications

  1. The page must have a header tag (eg. h1 tag).
  2. The page must have a list (eg. ul or ol tag and li tags)
  3. The page must have HTML tag to start and end the file properly.
  4. The page must be called "sc2.html"
  5. The page might have a body and head tags
  6. The page might have a title (title tag)
