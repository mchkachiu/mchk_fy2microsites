TLDR

Summary (TLDR = "Too long, didn't read")

In the "Power of Microsites" unit, you will learn some fundamentals around making websites.

Your main task will be to design or redesign 2-3 webpages that are effective in solving a real-life problem.

This page contains task instructions and resources to guide you through the unit.

see Unit Overview

Unit Aims

...Click to hide / unhide this section...

Main task: Microsite (website) development / webpage redesign for a local group, community event or business.

The student:

  • develops the requirements of a 2-3 page website for a local group by presenting the results of an inquiry into the need and analysis of the information gathered
  • develops ideas and creates layout sketches of the web pages, gathering feedback from the group to inform development. Identifies text, images and other media suitable to the purpose of the website, demonstrating an understanding of graphic interface design through the use of space. Presents a final solution that meets the stated needs
  • develops a plan for the creation of each page, including how the pages link, with a consistent layout and colour scheme. Edits the digital components to ensure they are in an appropriate format, explaining how the plans were altered in any way. Uses appropriate software to create the website, demonstrating excellent technical ability
  • interviews the group the website is designed for and gathers feedback on the effectiveness of the website. Uses this information to explain the success of the website, how it could be improved, and how it has an impact on the group.
Note: A website created by online template services or exported to web pages by other applications would not be a sufficiently sophisticated product/solution.
* Adapted From IB MYP Design Teacher Support Material (TSM) 2014

Project Task

...Click to hide / unhide this section...

All pupils should design and build a complete and effective 2-3 page mini website with appropriate media content.

It is possible to expand this project into a group project (2-3 pupils), consisting of up to 6 pages.

All pages should have an authentic (real-world) application, ie. the created sites should solve an existing real-world problem (they are not created around fictitious problems or contrived scenarios). While you are free to define your own problems and constraints for this project, the list below provides some good options for you to explore:

  • Create a microsite to support an NGO (event, program info, website section redesign)

  • Create a microsite for a school event / campaign / CCA / club

  • Redesign a section of a dated website

  • Build a website to streamline a simple yet common task

  • Build a microsite to teach or build awareness around a simple concept or topic

Unit Overview

There are two main strands in this unit.

Skills classes will be run on Monday and Tuesday lessons. These will focus on developing web development skills such as HTML and CSS.

Project Classes will run on all other days. In these classes you focus on developing your project through the design-cycle.

A3 Product Analysis
  • Attend Instructional Lesson: Product Analysis
  • Find existing webpages that solve similar problems to the problem you are looking at
  • Analyse these webpages. Consider:
    • MAP (Message, Audience, Purpose)
    • Visual Layout (Elements of Design)
    • Files, Media and Data
    • Content
    • Usability (navigation, CTAs, user experience)
  • What have you learned from analysing these websites? What conclusions can you make? How will it guide your project work.
