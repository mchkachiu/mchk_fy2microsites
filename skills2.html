Skills Challenge 3

Usually our first page in a website has the file name of “index.html”. In this challenge you will create a new file called “index.html” and create a simple list with links to the two pages you have already created (sc1.html and sc2.html). There should be a 3rd link in the list that takes users to a new Google search (www.google.com).

This file should be saved as index.html and saved in your "WWW_FY2..." Google folder.

Sample Page

Specifications

  1. The page must be saved as “index.html”.
  2. The page must have a title (<title> tag in the <head> section).
  3. The page must have a body and in the body there must be a list (eg. ul or ol tag and li tags)
  4. The list must contain links that take users to your sc1.html and sc2.html pages (use the <a href=””>...</a> tag)
  5. The list must contain a link to a Google search.
  6. The page must have an appropriate header (such as “Welcome” as <h1>)
  7. The page might have an image.


Skills Challenge 4

CSS (Cascading Style Sheets) are a way to format the elements in a website. It is a powerful way for web designers to set the style (such as the font, colour, size, appearance) of ALL <h1> tags or ALL paragraphs, or lists. CSS is often saved in another file, usually called “stylesheet.css”.

In this challenge you will copy a set sc4.html file, then create a new file called “stylesheet.css” and use that to change the formatting in the other files on your website. Your challenge is to make your sc4.html file look like the following sample page:

Sample Page

Follow the Instructions and Specifications below to complete the task.

All files should be saved in your "WWW_FY2..." Google folder.

You will need to know a little about CSS in order to complete this challenge, including how to select a HTML tag, how to select a class and how to select an ID.

Khan Academy "Intro to CSS" will help you learn CSS.

Instructions and Specifications

  1. Make a copy of this code and paste it into a new file called “sc4.html” in your "WWW_FY2..." Google folder.
  2. Now, in Brackets, make a new file and save it as "stylesheet.css" in the same folder as your sc4.html file.

  3. Now let's change our H2 headers.

    All H2 headers should:

    • have a font-size of 18px,
    • have a serif font-family,
    • have a grey colour font.

    In your stylesheet.css file, make those changes.

    Hint: try
    h2 {
    font-size: ...
    }
    to start

  4. Now let's change our paragraphs.

    All p tags should:

    • have a font-size of 15px,
    • have a san-serif font-family,
    • have a line-height of 22px,
    • have a #666 colour font.

    In your stylesheet.css file, make those changes.

    Hint: try
    p {
    font-size: ...
    }
    to start

  5. Next, let's make our main post heading image larger.

    The "post-head-image" ID should:

    • have a padding-top of 140px,
    • and a padding-bottom of 140px.

    This should give a 140px gap above and below any text in that div (box). This makes the div with id "post-head-image" much taller, displaying the background image.

    Hint: try
    #post-head-image {
    padding-top: ...
    }

  6. Let's make the main headline (which has an class of "pageTitle") white, larger and right-aligned.

    The "postTitle class should:

    • have a font-size of 4rem (4 times the font size relative to the browser),
    • have a shadow,
    • be transformed to uppercase (text-transform)
    • and be white./li>

    Hint: try
    .postTitle {
    text-shadow: ...
    }

  7. Extension (bonus):

    Fix up the footer.

    The footer should:

    • have a background color of #1a4b35 ,
    • have the navigation links displayed horizontally (Hint: see W3Schools page on navigation),
    • have the footer navigation text change colour (darken) when you hover over them (see the footer in this example)
